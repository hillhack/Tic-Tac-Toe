Requirements

    Node.js and npm: Make sure you have Node.js and npm (Node Package Manager) installed on your system. 
    Express: Express is a web application framework for Node.js. It provides a robust set of features for web and mobile applications.
    graphql: The graphql package is the core library for building GraphQL servers and clients in JavaScript.
    express-graphql: The express-graphql package provides a way to create a GraphQL HTTP server with Express.
    
 Steps:
 	Create a new directory for your project
 	Initialize a new npm project(Terminal command): npm init -y
	Install dependencies: npm install express graphql express-graphql
	Create a new file for your server:server.js
